Very basic example

We can simply make element dragable just selecting element and bounds 


Draggable.create('.cube', { bounds: '.container' });

We can change type to make object rotateble instead dragable 


Draggable.create('.cube', {
  bounds: '.container',
  type: 'rotation',
});

limiting by axis

To limit drag on x or y axis we can set the type as x or y 


Draggable.create('.cube', {
  type: 'y',
  bounds: '.container',
});
