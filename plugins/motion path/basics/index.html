Very basic example

To set a path we can use selector or we can use curves string, or even array of point coordinates 


gsap.to('.cube', {
  duration: 5,
  repeat: -1,
  yoyo: true,
  motionPath:
    'M.36,89.63C80.64,4.35,110.81-4.89, ...',
});

Example above uses curve string

Autorotate

Autorotate true, forces object to rotate along path curve 


gsap.to('.cube', {
  duration: 5,
  repeat: -1,
  yoyo: true,
  motionPath: {
    path: 'M.36,89.63C80.64,4.35,110.81-4.89,123.49,2.56c23.39,13.72-16.09,82.08,16.33, ...',
    autoRotate: true,
  },
});

start, end

We can limit where on path animation starts and where ends, by setting start and end values, where 0 means path start, 0.25 means 1/4 of path. We can go over and set end 1.5 for example, it means path will go full and then repeats again half way 


gsap.to('.cube', {
  duration: 5,
  repeat: -1,
  yoyo: true,
  motionPath: {
    path: 'M.36,89.63C80.64,4.35,110.81-4.89,123.49,2.56c23.39,13.72-16.09,82.08,16.33, ...',
    autoRotate: true,
    start: 0.33,
    end: 0.66,
  },
});
