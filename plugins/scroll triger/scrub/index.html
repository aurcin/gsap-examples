Scrub

Scrub makes animation go full cycle where start marker means animation start and end animation end, so scrolling 20% from start puts us to 1/5 of timeline 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top 80%',
              end: 'bottom 20%',
              scrub: true,
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });

Passing number in scrub instead of boolean value, gives as much time for naiamtion to chatch up. 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top 80%',
              end: 'bottom 20%',
              scrub: 1,
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });
