trigger and toggleActions

gsap.to('.cube', {
  scrollTrigger: {
    trigger: '.cube',
    toggleActions: 'restart pause reverse pause',
   },
   x: 400,
   rotation: 360,
  duration: 3,
});

toggleActions is combination of 4 keywords, where each keyword can be either:

  • play, basic, just plays when triggered
  • pause
  • resume
  • reverse
  • restart, restarts each time is triggered
  • reset
  • complete
  • none

Each of 4 positions means: "when enter" "when going past" "when returning in" "when we all the way back"
