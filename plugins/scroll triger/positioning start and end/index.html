Positioning start and end

By default scroll trigger starts as soon as element top gets into viewport. We can pass two values to start attribute, first means what part of element (top means element top) and second where at viewport it should appear, for example "top center" means when top part of element reach center of viewport 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top center',
              toggleActions: 'restart pause reverse pause',
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });

You can use keyword values "top center bottom" or relative values like pixels or percentages. Those values are relative to vievport/element top. 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: '30px 80%',
              end: 'bottom 20%',
              toggleActions: 'restart pause reverse pause',
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });

Attribute end has same controls mentioned earlier plus end can use relative to start expression like +=300 means 300px after start trigger 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: '30px 80%',
              end: '+=100% 300px',
              toggleActions: 'restart pause reverse pause',
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });

We can also use functions for start and end triggers 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top center',
              end: () =>
                `+=${document.querySelector('.cube').offsetHeight} 40%`,
              toggleActions: 'restart pause reverse pause',
              markers: true,
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });

We can also use different element as a end trigger, so we define endTrigger:'.other-cube' 


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top center',
              endTrigger:'.other-cube
              end:'bottom, center',
              toggleActions: 'restart pause reverse pause',
              markers: true,
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });
