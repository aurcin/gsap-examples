Pinning

We can pin elements by using pin:'element'


          gsap.to('.cube', {
            scrollTrigger: {
              trigger: '.cube',
              start: 'top 80%',
              end: 'bottom 20%',
              scrub: true,
              pin: '.cube',
              markers: true,
            },
            x: 400,
            rotation: 360,
            duration: 3,
          });
