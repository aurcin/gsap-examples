Callbacks

onEnter, onLeave, onEnterBack, onLeaveBack

There are various cllbacks we can use on scroll trigger like onEnter, onLeave, onenterBack, onLeaveBack 


          ScrollTrigger.create({
            trigger: '.cube',
            start: 'top center',
            end: 'bottom center',
            onEnter: () => console.log('entered'),
            onLeave: () => console.log('leaved'),
            onEnterBack: () => console.log('entered back'),
            onLeaveBack: () => console.log('leaved back'),
          });

onUpdate

There is another callback onUpdate which is fired on every update, and callback receives self as parameter, where we can reach attributes like progress 


          ScrollTrigger.create({
            trigger: '.cube',
            start: 'top center',
            end: 'bottom center',
            onUpdate: (self) => console.log('updating', self.progress),
          });

onToggle

Callback onToggle which is fired on every time it enters or leaves trigger, and callback receives self as parameter, where we can reach attributes like isActive 


          ScrollTrigger.create({
            trigger: '.cube',
            start: 'top center',
            end: 'bottom center',
            onToggle: (self) => console.log('toggled', self.isActive),
          });

toggleClass

toggleClass is fired when trigger changes, if enters it adds class if leaves it removes a class 


          ScrollTrigger.create({
            trigger: '.cube',
            start: 'top center',
            end: 'bottom center',
            toggleClass: 'active',
          });
