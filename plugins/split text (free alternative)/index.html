Here we use split-type npm library to split element text into characters, words or lines. It returs object with new elements and we can use those in gsap to animate 


const text = new SplitType('.text', { types: 'chars' });
gsap.from(text.chars, { opacity: 0, y: 20, ease: 'back(5)', stagger: 0.05 });
