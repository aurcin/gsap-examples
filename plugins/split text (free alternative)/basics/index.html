Here we use split-type npm library to split element text into characters, words or lines. It returs object with new elements and we can use those in gsap to animate 


const text = new SplitType('.text', { types: 'chars' });
gsap.from(text.chars, { opacity: 0, y: 20, ease: 'back(5)', stagger: 0.05 });

Breaking text into characters and animating them with gsap 


const text = new SplitType('.text', { types: 'words' });
gsap.from(text.words, {opacity: 0, rotation: -90, ease: 'back', stagger: 0.2 });

Breaking text into words and animating them with gsap 


const text = new SplitType('.text', { types: 'lines' });
gsap.from(text.lines, {
  opacity: 0,
  rotationX: -120,
  rotationY: 80,
  stagger: 0.3,
  transformOrigin: '50% 50% -150',
});
