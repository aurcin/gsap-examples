Basics

Timeline is a sequence of tweens where each tween starts after previous ends (by default) 


const timeline1 = gsap.timeline();

timeline1.to('.cube1', { duration: 1, x: 300 });
timeline1.to('.cube2', { duration: 1, x: 300 });
timeline1.to('.cube3', { duration: 1, x: 300 });

Animating 3 cubes, one after another each moving by 300px along x axis over 1 second
