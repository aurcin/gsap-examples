Keyframes

In gsap >3.9 we can use keyframes to animate multiple values at diferent time position in a single tween

percentage based keyframes


timeline.to('.cube', {
  keyframes: {
    '15%': { y: 0 },
    '50%': { y: -250, ease: 'power2.out' },
    '85%': { y: 0, ease: 'bounce.out' },
    '100%': { x: 630, ease: 'none' },
  },
  duration: 2,
});

This should be intuitive. 100% we set x to 630, what means our cube is moving 630px along x axis. Then we add some lifting from 15% of timeline to 50%, and after moving box back to 0 on y with bounce ease.
