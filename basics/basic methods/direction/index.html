Direction

We control animation flow by using gsap methods to, from and fromTo. We can also set some properties using set

to

Tweens from original properties to given values

gsap.to('.cube', { duration: 2, x: 300 });

Moves cube from current position by 300px along x-axis

from

Tweens from given values to original properties

gsap.from('.cube', { duration: 2, x: 300 });

Moves cube from 300px along x-axis from its original position to its original position

fromTo

Tweens from values to values

gsap.fromTo('.cube', { x: 150 }, { duration: 2, x: 300 });

Here we animate cube from 150px to 300px on x-axis

set

Sets values, without animations

gsap.set('.cube', { x: 300 });

Sets x property of cube to 300
