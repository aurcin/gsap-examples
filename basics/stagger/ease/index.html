Easing

We can use ease to controll stagger animation


gsap.to('.cube', {
  duration: 2,
  height: 250,
  repeat: -1,
  repeatDelay: 1,
  stagger: {
    amount: 5,
    ease: 'power2',
  },
});

Means there is half second between each animation. We have more controlls if we pass object insted of just number. To set interwals in object we use each property. We can also use amount instead each, amount means how much time we have for all staggers where each means delay between every animation
