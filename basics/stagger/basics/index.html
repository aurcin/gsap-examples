Basics

Stagger allows us to animate multiple objects, setting time intervals between each animation start. 

stagger:0.5

Means there is half second between each animation. We have more controlls if we pass object insted of just number. To set interwals in object we use each property. We can also use amount instead each, amount means how much time we have for all staggers where each means delay between every animation 


stagger: {
  each: 0.3
},

Here we set 300ms intervals

From

We can control where stagger is starting from. We can imagine stagger elements as array (or grid), so if we set from: 'end', - animation starts from last element 


gsap.to('.cube', {
  duration: 2,
  x: 600,
  stagger: {
    each: 0.3,
    from: 'center',
  },
});

In this example stagger starts from center


        gsap.to('.cube', {
          duration: 2,
          x: 600,
          stagger: {
            each: 0.3,
            from: 'edges',
          },
        });

In this example stagger starts from edges


        gsap.to('.cube', {
          duration: 2,
          x: 600,
          stagger: {
            each: 0.3,
            from: 3,
          },
        });

In this example stagger starts from element with index 3 (fourth element)
