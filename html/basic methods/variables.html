Variables

xPercent, yPercent

gsap.to('.cube', { duration: 2, xPercent: 500 });

Tells gsap to animate element with class cube moving it from 500% translated on x for its initial position

rotation, rotationX, rotationY

gsap.to('.cube', { duration: 3, rotation: 360 });

Rotates the cube by 360 deg over 3 seconds

gsap.to('.cube', { duration: 3, rotationX: 360 });

Rotates the cube by 360 deg on x axis over 3 seconds

gsap.to('.cube', { duration: 3, rotationY: 360 });

Rotates the cube by 360deg on y axis over 3 seconds

scale, scaleX, scaleY

gsap.to('.cube', { duration: 2, scale: 0.5 });

Scales the cube by half over 2 seconds

gsap.to('.cube', { duration: 2, scaleX: 0.5 });

Scales the cube by half on x axis over 2 seconds

gsap.to('.cube', { duration: 2, scaleY: 0.5 });

Scales the cube by half on y axis over 2 seconds

transformOrigin

gsap.set('.cube', { scale: 0.4, yPercent: -50, xPercent: 50 });
gsap.to('.cube', { duration: 3, rotation: 360, transformOrigin: '0% 100%' });

First we scale cube and move to starting postion, then we rotating it on bottom left corner, by seting transform origin

skewX, skewY

gsap.fromTo('.cube, { x: 100 }, { duration: 2, skewX: -45 });

First we move cube on it x by 100px, then we skew it by 45 deg on x axis

gsap.fromTo( '.cube', { scale: 0.5 }, { duration: 2, skewY: -45 } );

First we scale cube a bit then we skew it by 45 deg on y axis
