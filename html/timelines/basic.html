Timelines

Basic

const timeline = gsap.timeline();
timeline.to('.cube1', { duration: 1, x: 300 });
timeline.to('.cube2', { duration: 1, x: 300 });
timeline.to('.cube3', { duration: 1, x: 300 });

Tells gsap to animate 3 cubes, one after another each moving by 300px along x axis over 1second

timeline delay

timeline2.to( '.cube', { duration: 1, x: 355, ease: Bounce.easeOut, }, '-=.5' );

Only peace of code

Last parameter is used for delay, you can write number or string with relative change. In code above, we set delay a half second before previous animation ends.

timeline delay

timeline3.addLabel('end');
timeline3.to( '.cube', { x: 600, duration: 2, }, 'end' );

First line adds label to animation time, second line uses that label to start animation

timeline repeat

const timeline4 = gsap.timeline({ yoyo: true, repeat: -1 });

Only peace of code

We adding infinite repeats to timeline, playing animation forward then backward
