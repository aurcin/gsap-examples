Basic methods

to

gsap.to('.cube', { duration: 2, x: 300 });

Tells gsap to animate element with class cube moving it on x axis by 200px over 2 seconds

from

gsap.from('.cube', { duration: 2, x: 300 });

Tells gsap to animate element with class cube moving it to his initial colation, starting 200px away over 2 seconds

fromTo

gsap.fromTo('.cube', { x: 150 }, { duration: 2, x: 300 });

Tells gsap to animate element with class cube moving it from position x 150px to position x 300px over 2 seconds

set

gsap.set('.cube', { x: 300 });

Imediately sets x property of cube to 300, without nay animation
