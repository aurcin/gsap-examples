Units

default - pixels

gsap.to('.cube', { duration: 2, x: 300 });

Moves cube on x axis by its default units - pixels

Relative values '=+' and so on

gsap.to('.cube', { duration: 2, x: '+=300' });

Moves cube on x axis adding 300px to its current position

Other units as a string

gsap.to('.cube', { duration: 2, x: '20vw' });

Moves cube on x axis by 20 viewport width unit

Function

gsap.to('.cube', { duration: 2, x: () => Math.random() * 300 });

Moves cube on x axis by random number 0 to 299. px
